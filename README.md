## Run Locally

Clone the project

```bash
  git clone https://github.com/danielContreras28/test-aviatur-front
```

Go to the project directory

```bash
  cd test-aviatur-front
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm run start
```
